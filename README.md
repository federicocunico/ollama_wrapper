# Ollama wrapper

## Requirements:

- Python 3.10+
- Ollama

## Installation

In a terminal, run the following command:

```bash
cd <project_directory>
pip install poetry
poetry install
pip install .
```

Check installation:
```bash
python -c "import ollama_wrapper"
```

If the output is empty, the installation was successful.
